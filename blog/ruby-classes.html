This week is all about Classes in Ruby. A class, at its core like everything else in ruby, is an object that allows you to create new instances with methods built in. Methods that you define within said class. A class is essentially a blueprint, for what you want to build, your own class. For example if want to create a new class: Song


  class Song
    def initilaize(name, artist, duration)
      @name = name
      @artist = artist
      @duration = duration
    end
  end

Here we created a basic class Song, which contains a single method, initialize. Which is a special method in Ruby to do exactly what it says, initialize the new class we have created. To set up the new object Song state. In this example Song the initialize method takes three parameters, name, artist, and duration. We then need to store these values as instance variables (@name, @artist, @duration) within the object. This way we can use the variables anywhere else inside the class as well.

We can then call upon this Class (with three parameters) to make a new object. To test, we can call to the class. This will create newsong variable using the three parameters built from the Song class.

      
        newsong = Song.new('Bohemian Rhapsody', 'Queen', 360)
